# veda_ped_analyzer

`veda_ped_analyzer` is a GUI-based Python tool for analyzing **Potential Energy Distribution (PED)** data
generated by **VEDA** (`.ved`, `.dd2`). It can optionally correlate VEDA vibrational modes with
quantum-chemistry outputs from **ORCA** or **Gaussian**.

The main purpose of this software is to produce clear, publication-ready CSV tables
summarizing PED contributions for vibrational analysis.

## Inputs

Required:
- VEDA `.ved`
- VEDA `.dd2`
- Quantum-chemistry output:
  - ORCA: `.out`
  - Gaussian: `.log` or `.out`

Optional:
- `.fmu` (used for improved atom labeling if available)

## Outputs

- `*_PED_table.csv`  
  Vibrational-mode table including dominant PED contributions.
- `*_coordinates_lookup.csv`  
  Internal-coordinate list with dominant vibrational modes (`top_mode_terms`).

### Key columns in `*_PED_table.csv`

- `mode_veda`, `freq_veda`
- `mode_qc`, `freq_qc`, `delta_freq`, `abs_delta_freq`
- `irrep` (if available)
- `IR_intensity` (IR intensity from ORCA/Gaussian; 0 or blank if unavailable)
- `PED1_*` … `PED6_*`

## Running the program

```bash
python veda_ped_analyzer.py
```

On Windows, `veda_ped_analyzer.pyw` can be used to suppress the console window.

## Common Windows issue: Permission denied

If you encounter `[Errno 13] Permission denied` when saving CSV files, common causes include:
- The output CSV is opened in Excel (file lock)
- The output directory is write-protected (e.g., `C:\` root)
- Sync folders (OneDrive, etc.) restrict writes

Close Excel and choose a writable directory such as Desktop or Documents.
